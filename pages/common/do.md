# do

> This shell keyword is used with `for`, `while`, `select`, and `until` to mark the start of a loop.

- View documentation for the `while` keyword:

`tldr while`

- View documentation for the `while` keyword:

`tldr for`

- View documentation for the `while` keyword:

`tldr select`

- View documentation for the `while` keyword:

`tldr until`
